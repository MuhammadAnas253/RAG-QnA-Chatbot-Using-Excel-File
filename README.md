RAG Q&A Chatbot Using Excel File
A Retrieval-Augmented Generation (RAG) chatbot that allows users to upload an Excel file and ask natural language questions about the data. The system processes the spreadsheet content, converts it into vector embeddings, and uses a language model to generate intelligent answers based on the data.
This project demonstrates how structured data (Excel spreadsheets) can be integrated into an LLM-powered question-answering system.

**Project Overview
**
This application enables users to:
Upload an Excel (.xlsx) file
Extract and process spreadsheet data
Convert data into vector embeddings
Store embeddings in a vector database
Ask natural language questions about the dataset
Receive accurate answers generated by an LLM using retrieved context
The system follows a Retrieval-Augmented Generation (RAG) architecture, widely used in modern AI applications.

**RAG Workflow**
The system operates through the following pipeline:
User uploads an Excel file
The file is loaded using an Excel Loader
Data is converted into text documents
Documents are split into manageable chunks
Chunks are transformed into vector embeddings
Embeddings are stored in a vector store

**When the user asks a question:**
Relevant chunks are retrieved
Context is passed to the language model
The model generates a context-aware answer
